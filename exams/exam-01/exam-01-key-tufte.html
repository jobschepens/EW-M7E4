Introduction

We will use an existing dataset that contains speaking proficiency test scores on an official language exam. These speaking proficiency scores are taken from the State Exam Dutch as a Second Language as administred by the Dutch Board of Examinations. The data are available from https://zenodo.org/record/2863533. This dataset was published alongside a recent study on the effects of language similarity on speaking proficiency (Schepens, Hout, & Jaeger, 2020). The study used three different measures of linguistic similarity:

  • a lexical distance measure based on (Gray & Atkinson, 2003) using data that was retrieved from the the Indo-European Lexical Cognacy Database (IELEX),
  • a morphological similarity measure based on The World Atlas of Language Structures (WALS) (Dryer & Haspelmath, 2011), and
  • a phonological distance measure based on data from Phonetics Information Base and Lexicon (PHOIBLE) (PHOIBLE Online, 2014).

l1s <- stex %>% 
    group_by(L1) %>%

          score = mean(Speaking), 
          l = lex[1],

l1s <- l1s %>% gather(key = "Distance", value = "Value", l, p, m)

l1s <- l1s %>% mutate(fam2 = case_when(
  fam == "Indo-European" ~ "IE",
